## 기술적 이슈 및 해결

| 이슈 | 원인 | 해결 |
|------|------|------|
| `@tosspayments/payment-sdk` 없음 | 패키지명 변경됨 | Toss CDN 방식(window.TossPayments)으로 전환 |
| RTCSessionDescriptionInit 타입 오류 | Node.js는 브라우저 타입 미지원 | `any` 타입으로 교체 |
| Railway "Failed to fetch" | 원인 미상 | Render.com으로 플랫폼 변경 |
| Git "Author identity unknown" | git config 미설정 | user.email/name 전역 설정 |
| GitHub "repository not found" | 저장소 미생성 | GitHub에서 저장소 생성 후 push |
